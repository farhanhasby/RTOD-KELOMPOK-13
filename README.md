# RTOD (Real-Time Object Dimension Measurement)

Proyek pengukuran dimensi objek secara realtime menggunakan OpenCV dan PyQt5.

## Fitur
- Kalibrasi objek referensi
- Pengukuran panjang dan lebar objek
- GUI interaktif
- Screenshot hasil pengukuran

## File Utama
- RTOD_v1.py
- RTOD_v2.py
